Welcome to Drools
=================

Run the examples
----------------

It's easy to run the examples:
On Linux, Mac or Cygwin:
  examples/runExamples.sh
On Windows:
  examples\runExamples.bat


Run the examples in Eclipse
---------------------------

Open menu File, menu item Import, tree item Maven, tree item Existing Maven Projects, button Next
and select "examples/sources/" from the zip, button Finish.

In Package Explorer, right click "DroolsExamplesApp.java" under "org.drools.examples" package
and choose "Run As" -> "Java Application".


Run the examples in IntelliJ
----------------------------

Open menu File, menu item Open, select "examples/sources/pom.xml".

In Project Window, right click "DroolsExamplesApp" under "org.drools.examples" package
and choose "Run 'DroolsExampleApp...main()'".
