KIE API - Core Package

This package contains the core interfaces of the Apache KIE (Knowledge Is Everything) API, providing the foundation for building and executing business rules, processes, and decision services.

Key Components

KieServices

The central entry point and factory for all KIE services. This thread-safe singleton provides access to:

  • KieContainer - for loading and managing KIE modules
  • KieRepository - for managing KIE module artifacts
  • KieBuilder - for building KIE modules from resources
  • Configuration factories - for creating KieBase and KieSession configurations

KieBase

A repository of knowledge definitions (rules, processes, functions, type models). The KieBase is immutable and serves as a factory for creating KieSession instances where runtime data is managed.

KieBaseConfiguration

Configuration interface for customizing KieBase behavior at instantiation time. Supports property-based configuration for equality behavior, sequential mode, TMS, and more.

PropertiesConfiguration

Base interface for configuration classes, providing simple property get/set operations.

Getting Started

Basic usage example:

// Get the KieServices singleton
KieServices kieServices = KieServices.get();

// Create a KieContainer from the classpath
KieContainer kieContainer = kieServices.getKieClasspathContainer();

// Get a KieBase
KieBase kieBase = kieContainer.getKieBase();

// Create a KieSession
KieSession kieSession = kieBase.newKieSession();

// Insert facts and fire rules
kieSession.insert(myFact);
kieSession.fireAllRules();

// Clean up
kieSession.dispose();
